# Vishenka_talk
VAESCryptTalk Chat — клиент

Secure. Weird. Working.

Клиент для защищённого мессенджера с шифрованием, файлами, звонками и поддержкой бесед. 

## Быстрый старт

Скачайте всё содержимое этой папки (JAR + файлы .jks и .db).
Установите Java 17 (если ещё нет): Adoptium
Запустите клиент в терминале:

`bash
java -jar messenger.jar`

### Что умеет
Приватные сообщения
Групповые беседы
Сквозное шифрование (AES + RSA)
Отправка файлов
Голосовые звонки (если есть микрофон)
Список друзей
Закреплённые чаты
Статусы «онлайн/оффлайн»

## Структура папки

Все эти файлы должны лежать рядом с messenger.jar:

`text`
`clienttruststore.jks`   — сертификаты
`friends.db`        — список друзей
`pinned_chats.db`       — закреплённые чаты
Без них клиент может не запуститься или работать криво.

## Запуск из командной строки
`bash
cd /путь/к/папке/с/клиентом
java -jar messenger.jar`
Если клиент не запускается — проверьте:

- Версию Java (java -version должна показывать 17+)
- Наличие всех .jks и .db файлов
- Для тех, кто хочет собрать сам:
`bash
javac *.java
jar cfe messenger.jar Client *.class`
